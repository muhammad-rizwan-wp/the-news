# 📰 The News — Blog & Magazine

A modern and responsive **news/blog website template** built using **HTML, CSS & JavaScript**, based on a community Figma design.

---

## 💡 About The Project

This is a static news/blog website template created from a Figma UI design.  
The goal of this project is to transform a clean, professional layout from Figma into a fully responsive and interactive frontend using vanilla web technologies.

Key focuses:
- Clean typography & layout
- Responsive design for all screen sizes
- Interactive UI elements using JavaScript
- Easy-to-expand structure for blog features
